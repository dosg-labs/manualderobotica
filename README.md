# Enciclopédia de Robótica Educacional 🤖📚
### *Guia Interativo do Manual de Robótica da SEDUC-SP*

Seja bem-vindo(a) ao repositório do **Manual de Robótica**! 

Este projeto transforma o manual tradicional do kit de robótica da rede estadual de São Paulo em uma plataforma de pesquisa interativa e hipertextual. Ele foi desenhado para organizar a aprendizagem de forma dinâmica, servindo como um guia prático em tempo real dentro da sala de aula para alunos e professores.

🌐 **Acesse a plataforma ativa aqui:** [https://dosg-labs.github.io/manualderobotica/](https://dosg-labs.github.io/manualderobotica/)

---

## 🚀 Como Funciona a Enciclopédia?

Ao contrário de um PDF estático, esta plataforma funciona por meio de **verbetes e categorias integradas**, onde cada componente do kit físico está conectado a uma rede de conhecimento:

* 📖 **Conceitos & Verbetes:** Explicações diretas e pedagógicas sobre o funcionamento de cada item.
* 🎥 **Vídeos Tutoriais:** Links diretos para os vídeos desenvolvidos pela equipe de formação da rede.
* ⚙️ **Dados de Fabricante (Datasheets):** Especificações técnicas traduzidas de forma acessível para projetos práticos.
* 🏷️ **Categorização Inteligente:** Navegação fluida que associa sensores, atuadores, cabos e microcontroladores aos seus respectivos usos na cultura maker.

---

## 🏫 Aplicação em Sala de Aula

* **Para Estudantes:** Facilita a pesquisa autônoma durante a resolução de desafios. Se o aluno esqueceu como funciona um Sensor Ultrassônico, ele acessa o verbete, assiste ao vídeo da formação e vê exemplos de ligação em segundos.
* **Para Professores:** Serve como material de apoio rápido para fundamentação teórica e mediação pedagógica durante as aulas práticas.
